# Once - REDAXO-Addon für die einmalige Ausführung von PHP

Once führt das in PHP aus, das du ihm schreibst. Für alle, die einmalig PHP auszuführen haben, um bspw. etwas zu testen, anzupassen oder auszuprobieren.
